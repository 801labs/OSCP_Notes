SMB Client

Listing Directories from port 445

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
smbclient -L //10.10.10.100
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Using a Kerberos Ticket

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
smbclient -k -L //10.10.10.100
